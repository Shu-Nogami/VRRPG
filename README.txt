作品説明
特にストーリーはない一般的なVR RPGになります。
プレイヤーは右手に剣を持ち、左手から魔法を打つことができます。
魔法は左手のボタンを押した回数によって変化し、ものによって効果は様々です。
（コマンドの組み合わせは魔法コマンド意味一覧をご覧ください）
また、魔法一つ一つに取得条件があり、プレイヤーレベルや倒した敵の種類などの条件があります。

進め方
はじめは道中の敵を倒しながらレベルを上げていき、魔法を取得していきます。
ある程度のレベルまで行った後はマップの端にいる中ボスを３体倒します。
その後マップ中央に行きラスボスを倒して終了です。
